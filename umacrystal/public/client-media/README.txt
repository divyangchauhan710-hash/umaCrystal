This folder is for uploading the client's original product images and videos.

Usage:
1. Place the original MP4 videos or images in this directory.
2. In the `data/products.json` file, update the `url` for the media to point to these files instead of external links.
   - For example: `"url": "/client-media/your-video.mp4"`
   - For images: `"url": "/client-media/your-image.jpg"`
